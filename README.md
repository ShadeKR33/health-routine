# 💪 헬스 트래커

개인 맞춤형 운동 루틴 및 식단 관리 웹앱

🔗 **[바로가기](https://[유저명].github.io/health-routine)**

---

## 📋 기능

- **🏠 홈** — 현재 체중, 턱걸이/푸쉬업 최대 기록, 이번 주 달성률 한눈에 확인
- **🏋️ 운동 루틴** — 요일별 상세 운동 루틴 (세트, 횟수, 팁 포함)
- **🍳 식단** — 요일별 식사 시간 및 메뉴, 질렸을 때 대체 메뉴 제공
- **📝 주간 기록** — 체중, 턱걸이, 푸쉬업 기록 저장
- **📊 히스토리** — 주차별 기록 누적 확인

---

## 🗓️ 운동 스케줄

| 요일 | 운동 | 시간 |
|------|------|------|
| 월 | 가슴 + 삼두 | 16~19시 중 선택 |
| 화 | 등 + 이두 | 16~19시 중 선택 |
| 수 | 😴 휴식 | — |
| 목 | 어깨 + 하체 | 16~19시 중 선택 |
| 금 | 가슴 + 삼두 | 16~19시 중 선택 |
| 토 | 등 + 이두 | 오전 11시 |
| 일 | 😴 휴식 | — |

## 🍳 식단 원칙

- 하루 단백질 목표 **100g**
- 3끼 + 간식 구조
- 운동 후 30분 이내 단백질 보충 필수
- 공복 운동 금지
